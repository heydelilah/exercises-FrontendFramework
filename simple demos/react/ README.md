voteit
==========

## 描述

一个使用 React.js 做的简易投票系统，可新增项目、为项目投票，并实时渲染投票结果。


## 涉及

- Gulp -task runner
- Browserify - 模块化
- Reactify －jsx transformer
- Firebase -database
